Dual Recommendation Engines

"Bookmarked" utilizes two distinct methods to provide rich, contextual recommendations, showcasing different technical approaches to the same problem.

ML-Powered "For You" Page

A classic collaborative filtering model trained on the entire user ratings dataset. It identifies users with similar tastes to predict what you might like next.

  • PythonUses Scikit-learn
  • FastAPIServed via a dedicated microservice
  • Use CaseBroad, personalized discovery

Graph-Powered "Readers Also Liked"

A real-time recommendation based on graph traversal. It instantly finds books that were highly rated by other users who also loved the book you're currently viewing.

  • Neo4jPowered by a Cypher query
  • Real-TimeInstant results based on relationships
  • Use CaseContextual, item-based discovery
